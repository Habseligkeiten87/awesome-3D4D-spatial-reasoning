2D → 3D → 4D Technical Evolution

The Shortcut Bias Critique

快捷偏差批判

Current models exploit semantic cues and object names rather than true spatial understanding. SURPRISE3D reveals the gap between claimed and actual spatial reasoning ability.

当前模型利用语义线索和物体名称而非真正的空间理解。SURPRISE3D 揭示了宣称的空间推理能力与实际表现之间的鸿沟。
