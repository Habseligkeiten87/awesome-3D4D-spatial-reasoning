Core Narratives

Cognitive Science → AI

From Craik's mental models and Tolman's cognitive maps to modern world models and latent representations. How classical cognitive science inspires contemporary machine learning architectures.

2D → 3D → 4D Evolution

Technical progression from 2D image understanding (LISA) through 3D scene comprehension (MLLM-For3D) to dynamic 4D spatial reasoning. Each layer adds spatial and temporal complexity.

The Shortcut Bias Critique

SURPRISE3D reveals fundamental gaps in current spatial reasoning. Models exploit 2D shortcuts rather than learning true 3D understanding, challenging assumptions about machine spatial intelligence.
